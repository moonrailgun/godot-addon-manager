# version

Display gdam version information.

## Usage

```bash
gdam version
```

## Description

The `version` command displays the current version of gdam installed on your system.

## Examples

```bash
gdam version
```

Output:
```
gdam 0.1.0
```

## Notes

- Useful for troubleshooting and reporting issues
- Check for updates by comparing with the latest release on the repository
